# 网络小说阅读器

一个功能完整的网络小说阅读器，支持获取小说封面、作者、简介、章节、内容、评分等功能，支持本地部署和远程访问。

## 功能特性

- 📚 **小说管理**: 显示小说封面、作者、简介、评分
- 📖 **章节浏览**: 查看小说章节列表，逐章阅读
- 🔍 **搜索功能**: 按标题或作者搜索小说
- 📱 **响应式设计**: 支持桌面和移动设备访问
- 🌐 **远程访问**: 支持局域网和互联网访问
- 💾 **本地存储**: 使用SQLite数据库存储小说数据

## 安装依赖

```bash
pip install flask requests
```

## 快速开始

```bash
# 运行应用
python novel_reader.py

# 应用将在 http://localhost:5000 启动
```

## 访问方式

- **本地访问**: http://localhost:5000
- **局域网访问**: http://[你的IP地址]:5000
- **远程访问**: http://[你的公网IP]:5000

## 使用说明

1. 首次运行会自动创建示例数据
2. 在主页面可以浏览所有已添加的小说
3. 点击小说卡片可查看详细信息和章节列表
4. 点击章节可在线阅读具体内容

## 数据管理

- 小说数据存储在 `novels.db` SQLite数据库中
- 封面图片存储在 `covers/` 目录
- 网页模板存储在 `templates/` 目录

## 扩展功能

可以通过API接口扩展更多功能：
- `/api/novels` - 获取所有小说列表
- `/api/novel/<id>` - 获取小说详情
- `/api/chapter/<id>` - 获取章节内容

## 部署建议

为了更好的性能和安全性，建议在生产环境中：
- 使用反向代理（如Nginx）
- 配置SSL证书
- 设置防火墙规则
- 定期备份数据库
