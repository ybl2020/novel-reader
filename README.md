# 网络小说阅读器

一个功能完整的网络小说阅读器，支持获取小说封面、作者、简介、章节、内容、评分等功能，支持本地部署和远程访问。**特别优化了移动端阅读体验！**

## 功能特性

- 📚 **小说管理**: 显示小说封面、作者、简介、评分
- 📖 **章节浏览**: 查看小说章节列表，逐章阅读
- ⏭️ **章节导航**: 快速切换上一章/下一章
- 🔍 **搜索功能**: 按标题或作者搜索小说
- 📱 **移动端优化**: 完美适配手机和平板设备
- 💻 **响应式设计**: 支持桌面和移动设备访问
- 🌐 **远程访问**: 支持局域网和互联网访问
- 💾 **本地存储**: 使用SQLite数据库存储小说数据

## 安装依赖

```bash
pip install flask requests
```

## 快速开始

### 方法一：使用启动脚本（推荐）
```bash
./start.sh
```

### 方法二：直接运行
```bash
python3 server.py
```

应用将在 http://localhost:8080 启动

## 访问方式

- **本地访问**: http://localhost:8080
- **局域网访问**: http://[你的IP地址]:8080
- **手机访问**: 确保手机和电脑在同一WiFi，访问 http://[你的IP地址]:8080

## 📱 移动端使用技巧

1. **手机横屏阅读**: 支持横屏模式，布局自动优化
2. **章节快速翻页**: 使用顶部或底部的"上一章/下一章"按钮
3. **搜索小说**: 搜索框自动适配手机屏幕
4. **触摸友好**: 所有按钮和卡片都优化了触摸点击体验

## 使用说明

1. 首次运行会自动创建示例数据
2. 在主页面可以浏览所有已添加的小说
3. 点击小说卡片可查看详细信息和章节列表
4. 点击章节可在线阅读具体内容

## 数据管理

- 小说数据存储在 `novels.db` SQLite数据库中
- 封面图片存储在 `covers/` 目录
- 网页模板存储在 `templates/` 目录

## 扩展功能

可以通过API接口扩展更多功能：
- `/api/novels` - 获取所有小说列表
- `/api/novel/<id>` - 获取小说详情
- `/api/chapter/<id>` - 获取章节内容

## 部署建议

为了更好的性能和安全性，建议在生产环境中：
- 使用反向代理（如Nginx）
- 配置SSL证书
- 设置防火墙规则
- 定期备份数据库
