# 📱 Novel Reader 演示指南

## 快速开始

### 1️⃣ 启动服务器

```bash
cd /Users/pollychen/.openclaw/workspace/novel-reader
./start.sh
```

你会看到类似输出：
```
🚀 正在启动网络小说阅读器...

📱 移动端优化版本
✨ 新功能：上一章/下一章导航

📍 访问地址：
   本地: http://localhost:8080
   局域网: http://192.168.1.100:8080

💡 用手机扫描下方二维码访问：
   http://192.168.1.100:8080

按 Ctrl+C 停止服务
```

### 2️⃣ 在手机上访问

1. 确保手机和电脑连接到**同一个WiFi**
2. 在手机浏览器输入显示的局域网地址
3. 例如：`http://192.168.1.100:8080`

## 🐛 常见问题

### Q1: 手机访问不了？
**A**: 确保手机和电脑在同一WiFi，检查防火墙设置

### Q2: 页面显示不完整？
**A**: 刷新页面，清除浏览器缓存

### Q3: 按钮点击无反应？
**A**: 确保 JavaScript 已启用

### Q4: 字体太小/太大？
**A**: 当前已优化为最佳大小，后续可添加字体设置
